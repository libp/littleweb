# 值得读读-web端 [![Build Status](https://www.travis-ci.org/libp/littleweb.svg?branch=master)](https://www.travis-ci.org/libp/littleweb)

## 介绍
* 基于vue2.0+elementUI+axios

## 已实现页面
* 首页获取推荐文章
* 随机页面获取随机文章
* 关于页面

## 安装使用
* 先启动zz工程提供API接口
* npm run dev 即可

## 相关工程介绍
* SpiderScrapyArticle，基于scrapy的最简爬虫工程
* zzLite，微信小程序源码
* littleweb，基于vue的前段web应用
* zz，基于springboot4.0+mybatis实现前后端分离工程
